# 🎬 Netflix Recommendation System

This project is a movie recommendation system built using Python and the MovieLens dataset.

## 📌 Features
- Recommends movies based on user similarity
- Also suggests similar movies (like Netflix)
- Built using collaborative filtering

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## 📊 Dataset
MovieLens dataset

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Upload movies.csv and ratings.csv
3. Run all cells
